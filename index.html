Craterstats

Craterstats is a program created by the United States Geological Survey (USGS). It is a tool to analyze and plot crater count data for planetary surfacing. The current version is developed by Greg Michael.

My contribution to this program is through my Senior Capstone which is sponsored by Trent Hare of the USGS. Me and a team of 4 other classmates are tasked with creating a Graphical User Interface (GUI) for the current version of craterstats. This current version is a python command line application. We have decided to make the GUI using the flet python library which is an extension of flutter. I have developed the current prototype which can be found in the GitHub Repo listed below. More information and technical documentation about this project can be found at the website below.

CarGame

CarGame is a game developed by me and 4 classmates for an end of semester project in our Virtual Worlds class. Virtual Worlds is a class to teach about Unity and C# development, while also teaching techniques of base game development.

My contribution to this game is developing the car mechanics as well as some basic small details around the game.

The next semester I took an independent study course with Professor Ashish Amresh. During this study it was my goal to upgrade the game by making it co-op multiplayer and then online multiplayer. After these new versions were created, I then had to recreate the game using Unity's Entity Component System (ECS). After converting the base game to ECS, another student, Siddharth Subramanian, and I collaborated on making the newly ECS game into a multiplayer game. Side Quests

Side Quests is a website that 5 of my classmates and I developed during our sophomore year for our Software Engineering class. The website was designed to be a task tracking website that would reward points and "experience points" once a task is complete. The user would also be able to compare levels and rewards with their friends to entice them to complete tasks.

My contribution to this project was to create the profile and rewards page. On this page both the information of the user that is logged in is displayed and the rewards they have received for the tasks completed.

While the website itself is no longer active the GitHub page with all the code still is.
